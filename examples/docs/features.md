# 功能特性说明

本文档详细说明移动端和桌面端原型的功能特性。

## 🎯 核心功能

### 1. 会话管理

#### 移动端
- **会话列表**：全屏显示，支持滚动
- **会话切换**：点击会话卡片进入聊天
- **返回导航**：顶部返回按钮回到列表
- **项目切换**：左侧抽屉选择项目
- **当前标识**：显示"当前"徽章

#### 桌面端
- **会话列表**：左侧边栏固定显示
- **会话切换**：点击会话项，中间区域更新
- **并行视图**：列表和聊天同时可见
- **项目切换**：顶部下拉选择器
- **高亮显示**：当前会话蓝色高亮

```
┌─────────────────────┐
```

#### 桌面端
```
┌────────┬──────────────────┬────────┐
│ 会话1  │  第一卷创作       │ 工作区  │
│ 会话2  ├──────────────────┤        │
```

**共同特性**：
- ✅ 用户/AI 消息区分
- ✅ 消息气泡样式
- ✅ 工具调用卡片
- ✅ 实时输入
- ✅ Enter 发送
- ✅ 自动滚动

### 3. Agent 配置

#### 移动端流程
```
我的页面
  └─ 点击 "Agent"
      └─ Agent 列表（全屏）
          └─ 点击 Agent
              └─ Agent 编辑器（全屏）
                  ├─ 基本信息
                  ├─ 模型配置
                  ├─ 提示词
                  └─ [保存] [取消]
```

#### 桌面端流程
```
左侧边栏
  └─ 点击 "Agent" 标签
      └─ Agent 列表（侧边栏）
          └─ 点击 Agent
              └─ 编辑器（中间区域）
                  ├─ 基本信息
                  ├─ 模型配置
                  ├─ 提示词
                  └─ [保存] [取消]
```

**配置项**：
- 📝 Agent 名称
- 🤖 模型选择
- 📊 采样参数
- 💬 系统提示词
- 🔧 工具配置

### 4. 文件工作区

#### 移动端
- **位置**：聊天界面的"会话工作区" Tab
- **显示**：全屏文件树
- **操作**：点击文件查看/编辑
- **切换**：Tab 切换回聊天

#### 桌面端
- **位置**：右侧边栏"工作区" Tab
- **显示**：固定显示，320px 宽
- **操作**：点击文件查看/编辑
- **并行**：与聊天同时可见

**文件树功能**：
- 📁 文件夹展开/折叠
- 📄 文件图标区分
- 🔍 文件路径显示
- ✏️ 文件操作菜单
- 🎯 纳入规则显示

### 5. 会话日志

#### 移动端
```
会话操作抽屉
  └─ 点击 "会话日志"
      └─ 日志页面（全屏）
          ├─ 工具调用记录
          ├─ 检查点列表
          └─ 回滚操作
```

#### 桌面端
```
右侧边栏
  └─ 点击 "日志" Tab
      └─ 日志面板（固定）
          ├─ 工具调用记录
          ├─ 时间戳
          └─ 状态标识
```

**日志信息**：
- 🔧 工具名称
- ✅ 执行状态（成功/失败）
- 📝 操作摘要
- ⏰ 时间戳
- 🔄 检查点 ID

### 7. 主题系统

#### 浅色主题
```css
背景：#F5F5F7 (浅灰)
表面：#FFFFFF (白色)
文字：#1D1D1F (深灰)
主色：#007AFF (蓝色)
```

#### 深色主题
```css
背景：#1C1C1E (深灰)
表面：#2C2C2E (中灰)
文字：#F5F5F7 (浅灰)
主色：#0A84FF (亮蓝)
```

**切换方式**：
- 移动端：顶部右侧按钮
- 桌面端：左上角按钮
- 自动保存：localStorage
- 平滑过渡：0.2s

## 🎨 交互细节

### 移动端交互

#### 触摸优化
- **最小触摸目标**：44x44px
- **按钮间距**：至少 8px
- **滑动手势**：支持左右滑动（可扩展）
- **下拉刷新**：支持下拉刷新（可扩展）

#### 导航模式
- **底部导航**：对话、Agent、我的
- **顶部返回**：层级导航返回
- **抽屉菜单**：项目列表、会话操作
- **底部弹出**：操作菜单

#### 动画效果
- **页面切换**：滑动动画
- **抽屉打开**：从侧边滑入
- **弹出菜单**：从底部滑入
- **Toast 提示**：淡入淡出

### 桌面端交互

#### 鼠标优化
- **悬停效果**：背景色变化
- **点击反馈**：轻微缩放
- **焦点样式**：蓝色边框
- **右键菜单**：上下文菜单（可扩展）

#### 导航模式
- **侧边栏导航**：会话、Agent、设置
- **Tab 切换**：右侧面板切换
- **面包屑**：路径导航（可扩展）
- **快捷键**：键盘快捷键（可扩展）

#### 动画效果
- **悬停动画**：平滑过渡
- **视图切换**：淡入淡出
- **Toast 提示**：从底部滑入
- **加载状态**：骨架屏（可扩展）
