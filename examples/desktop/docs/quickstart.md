# 快速开始指南

## 🚀 5 分钟上手桌面端原型

### 第一步：打开原型

选择以下任一方式：

**方式 A：直接打开（最简单）**
```bash
# 双击 index.html 文件
# 或在命令行中：
start index.html  # Windows
open index.html   # macOS
```

**方式 B：本地服务器（推荐）**
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 然后访问
http://localhost:8000/index.html
```

### 第二步：探索界面

#### 左侧边栏（导航区）
1. **顶部**：点击 🌙 图标切换深色/浅色主题
2. **导航标签**：
   - 📱 会话：查看和切换会话
   - 🤖 Agent：管理 AI 助手
   - ⚙️ 设置：配置系统设置
3. **项目选择器**：点击可切换项目（演示功能）

#### 中间主内容区
1. **聊天界面**：
   - 查看消息历史
   - 在底部输入框输入消息
   - 点击发送按钮或按 Enter 发送
2. **工具调用**：查看 AI 执行的操作
3. **顶部操作**：切换模型、会话菜单等

#### 右侧边栏（工具区）
1. **工作区标签**：查看当前会话的文件
2. **日志标签**：查看工具执行记录

### 第三步：尝试功能

#### 发送消息
1. 点击底部输入框
2. 输入 "你好，请帮我写一段代码"
3. 按 Enter 或点击发送按钮
4. 查看 AI 回复（演示模式）

#### 切换会话
1. 在左侧边栏点击不同的会话
2. 观察中间区域内容变化
3. 查看右侧工作区文件更新

#### 切换主题
1. 点击左上角的主题按钮
2. 观察整个界面颜色变化
3. 主题偏好会自动保存

#### 查看 Agent
1. 点击左侧 "Agent" 标签
2. 查看可用的 AI 助手列表
3. 点击任一 Agent 进入编辑模式
4. 点击 "取消" 返回聊天界面

#### 查看日志
1. 点击右侧 "日志" 标签
2. 查看工具执行历史
3. 观察成功/失败状态

### 第四步：查看主题演示

打开 `theme-demo.html` 查看完整的组件和主题效果：

```bash
start theme-demo.html  # Windows
open theme-demo.html   # macOS
```

## 🎨 自定义主题

### 修改颜色
编辑 `css/styles.css` 中的 CSS 变量：

```css
:root {
    --primary-color: #007AFF;  /* 改为你喜欢的颜色 */
    --success-color: #34C759;
    /* ... 其他颜色 */
}
```

### 调整布局
修改侧边栏宽度：

```css
:root {
    --sidebar-width: 280px;      /* 左侧边栏 */
    --right-sidebar-width: 320px; /* 右侧边栏 */
}
```

## 🔧 常见问题

### Q: 为什么有些功能点击没反应？
A: 这是一个 UI 原型，部分功能显示 "待实现" 的 Toast 提示。完整功能需要后端支持。

### Q: 如何重置主题？
A: 打开浏览器开发者工具（F12），在 Console 中输入：
```javascript
localStorage.removeItem('nm-desktop-theme');
location.reload();
```

### Q: 消息发送后没有真实回复？
A: 这是演示模式，AI 会返回一个模拟回复。真实功能需要连接后端 API。

### Q: 可以在移动设备上使用吗？
A: 桌面端原型针对大屏幕优化。移动设备请使用 `../mobile/` 目录下的移动端原型。
