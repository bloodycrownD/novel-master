# 更新日志

## [2.0.0] - 2024 - 主题系统重大更新

### ✨ 新增功能

#### 主题系统
- 🎨 **双主题支持**：完整的浅色和深色主题
- 🔄 **一键切换**：顶部导航栏主题切换按钮
- 💾 **自动保存**：主题偏好持久化到 localStorage
- 🌓 **平滑过渡**：所有元素 0.2s 过渡动画

#### 视觉优化
- ✨ **改进的颜色系统**
  - 新增 `--primary-hover` 悬停色
  - 新增 `--bg-secondary` 次要背景色
  - 新增 `--surface-elevated` 提升表面色
  - 新增 `--text-tertiary` 三级文本色
  - 新增 `--border-light` 浅色边框
  - 新增 `--overlay` 遮罩层颜色

- 🎯 **分层阴影系统**
  - `--shadow-sm`: 轻微阴影
  - `--shadow`: 标准阴影
  - `--shadow-lg`: 大阴影

- 🔵 **圆角优化**
  - 按钮：10px
  - 卡片：16px
  - 模态框：20px
  - 输入框：10-22px

#### 交互增强
- 🖱️ **悬停效果**：所有可交互元素
- 👆 **点击反馈**：scale(0.95-0.98) 动画
- 🎯 **焦点状态**：2px 蓝色轮廓
- ⚡ **过渡动画**：统一 0.2s 缓动

#### 特效增强
- 🌫️ **毛玻璃效果**
  - 顶部导航栏
  - 底部导航栏
  - 抽屉背景
  - 底部弹出菜单背景
  - Toast 提示

- 🎭 **动画效果**
  - 页面切换：cubic-bezier(0.4, 0, 0.2, 1)
  - 元素悬停：translateY(-2px)
  - 按钮点击：scale(0.98)
  - 未保存指示：pulse 动画

#### 无障碍支持
- ♿ **焦点可见性**：所有交互元素
- 🎬 **减少动画**：支持 `prefers-reduced-motion`
- 🔆 **高对比度**：支持 `prefers-contrast: high`
- 📱 **触摸优化**：最小 44px 触摸目标
