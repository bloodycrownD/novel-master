# 工作区与协作 — 功能清单（UI 设计用）

> **用途**：供 UI/UX 设计使用。只描述用户可见的能力、界面与状态，不涉及技术实现、框架或存储方式。  
> **范围**：项目与对话、AI 对话、虚拟文件、工作树、日志与检查点、Agent 配置、服务商、扩展设置。

---

## 1. 全局导航与信息架构

### 1.1 主导航

| 入口 | 说明 |
|------|------|
| 对话 | 当前会话的聊天主界面 |
| 文件 | 当前上下文下的虚拟文件浏览与管理 |
| 项目 | 项目列表、切换、项目级管理（可选独立 Tab 或抽屉） |
| 我的 | 服务商、Agent 模板、扩展设置、开发/调试入口（可选） |

### 1.2 全屏栈

- 从 Tab 或对话内入口 **压栈进入** 二级全屏（编辑文件、工作树、日志详情等）。
- 支持 **返回** 上一级；存在未保存内容时 **离开确认**。
- 底栏 Tab 在部分全屏场景下 **隐藏**。

### 1.3 通用反馈

| 能力 | 说明 |
|------|------|
| Toast | 短提示（成功、失败、进行中） |
| 确认对话框 | 破坏性操作、覆盖导入、删除等 |
| 输入对话框 | 重命名、多字段表单（如目录策略） |
| 底部 Sheet | 新建文件/目录、操作菜单、筛选等 |
| 侧滑抽屉 | 项目/会话列表（可选） |

## 3. AI 对话

### 3.1 对话顶栏 / 元信息

| 元素 | 说明 |
|------|------|
| 会话标题 | 可点击进入会话设置或重命名（可选） |
| Agent 名称 | 当前使用的 Agent 配置 |
| 模型 | 当前选用的模型 |
| 步数 / 轮次上限 | 多步 Agent 时的上限展示 |
| Token 用量 | 已用/估算上下文（可选） |
| 入口：改 Agent | 跳转 Agent 配置 |
| 入口：改模型 | 跳转模型选择 |

### 3.2 消息流

| 类型 | 说明 |
|------|------|
| 用户消息 | 用户发送的文本（及可选附件，若产品支持） |
| 助手消息 | AI 回复正文 |
| 工具调用卡片 | 每次工具调用的名称、状态（进行中/成功/失败）、摘要；可展开查看参数与结果详情 |
| 流式生成指示 | 正在生成时的条形或动画提示 |
| 历史加载 | 上滑加载更早消息（可选） |

### 3.3 会话操作抽屉（会话内顶栏 ☰）

| 入口 | 说明 |
|------|------|
| 真实提示词 | 全屏只读预览当前发给模型的真实提示词 |
| 会话日志 | 全屏统一时间线：工具执行与检查点创建按时间倒序同列展示 |

> 会话列表子视图下，顶栏 ☰ 打开 **项目抽屉**（非本表）。聊天输入区 **无** Chip 快捷入口；工作树通过 **会话工作区** Tab 与文件列表规则完成。

### 3.4 输入区

| 功能 | 说明 |
|------|------|
| 多行输入 | 支持换行 |
| 发送 | 提交消息并触发 AI |
| 停止生成 | 流式过程中中止（可选） |
| 禁用态 | 生成中或未选模型时不可发送 |

## 4. 虚拟文件（文件 Tab / 工作区）

### 4.1 作用域切换（若产品暴露多域）

| 作用域 | 用户理解 |
|--------|----------|
| 全局模板 | 全应用共享的模板目录 |
| 项目模板 | 仅属于当前项目的模板目录 |
| 会话工作区 | 当前对话专属的文件树（根为 `/`） |

> 设计时需明确：手机端默认展示哪一层、何时展示切换器。

### 4.2 文件浏览（列表模式）

| 功能 | 说明 |
|------|------|
| 当前路径展示 | 面包屑或单行路径，可点击上级 |
| 返回上级 | 从子目录回到父目录 |
| 目录列表 | 文件夹图标、名称；单击进入 |
| 文件列表 | 文件图标、名称；双击或菜单「打开」 |
| 行状态指示（工作树相关） | 见 §5 — 纳入方式、规则开关、展示结果的可视标记 |
| 行操作菜单 | 打开、切换纳入状态、重命名、删除、目录幻灯片等 |
| 工具栏操作 | 新建文件、新建文件夹、导入/导出压缩包（若支持）、应用目录策略等 |
| 空目录 | 空状态与新建引导 |

### 4.3 阅读模式

| 功能 | 说明 |
|------|------|
| 渲染正文 | 对 Markdown 等可渲染格式显示排版结果 |
| 全屏阅读 | 沉浸式阅读（可选） |
| 返回列表 | 回到浏览列表 |

### 4.4 编辑模式

| 功能 | 说明 |
|------|------|
| 源码编辑 | 等宽文本区 |
| 行号 | 左侧行号栏，与正文滚动同步 |
| 预览切换 | 源码 ↔ 渲染预览（仅对支持预览的类型，如 Markdown） |
| 未保存标记 | 标题或工具栏上的「未保存」提示 |
| 保存 | 写入当前文件 |
| 保存中 | 按钮禁用或加载态 |
| 离开确认 | 未保存时返回/关页需确认 |
| 检查点选择（可选） | 编辑顶栏选择历史检查点并回滚到该版本 |

### 4.5 目录幻灯片

| 功能 | 说明 |
|------|------|
| 连读目录下多个文件 | 左右切换上一篇/下一篇 |
| 显示当前序号 | 如 2/5 |
| 退出 | 回到列表 |

### 4.6 新建与重命名

| 功能 | 说明 |
|------|------|
| 新建文件 | 输入名称（及可选初始内容） |
| 新建文件夹 | 输入名称 |
| 重命名 | 文件或文件夹，校验重名 |

### 4.7 删除

| 功能 | 说明 |
|------|------|
| 删除文件 | 确认后删除 |
| 删除文件夹 | 非递归 / 递归删除需区分提示 |

### 4.8 导入 / 导出（若支持）

| 功能 | 说明 |
|------|------|
| 导出为压缩包 | 下载当前工作区快照 |
| 从压缩包导入 | 选择文件；**覆盖** 当前工作区需强警告 |
| 进行中 | 导入/导出过程禁用重复操作 |

## 5. 虚拟工作树

> 工作树决定「哪些文件以何种形式」进入 AI 可见的上下文摘要，不等同于文件是否存在。

### 5.1 预览

| 功能 | 说明 |
|------|------|
| 展示合成文本 | 只读、等宽、可滚动；内容为按规则拼接后的 `<file>` 块等格式 |
| 实时反映规则 | 在文件中改规则后，预览区内容应更新 |

### 5.2 文件夹：规则状态

| 状态 | 含义 |
|------|------|
| 规则·开 | 本目录纳入规则参与计算子文件 |
| 规则·关 | 子文件中「自动」项视为不展示 |
| 根目录 | 恒为「规则·开」，不可关闭 |

| 操作 | 说明 |
|------|------|
| 切换规则开/关 | 在文件夹行或详情中切换（根目录除外） |

### 5.3 文件：纳入方式（用户可切换）

| 方式 | 含义 |
|------|------|
| 自动 | 由父目录规则计算最终展示 |
| 展示 | 强制纳入，展示为全文 |
| 隐藏 | 强制不纳入 |

| 操作 | 说明 |
|------|------|
| 切换纳入方式 | 在三态间循环（如通过行菜单或图标） |

### 5.4 文件：显示状态（结果，只读理解）

| 状态 | 含义 |
|------|------|
| 不展示 | 不出现在工作树文本中 |
| 全内容 | 展示完整正文 |
| 文件头 | 仅展示文首元数据区（如 YAML front matter） |
| 文件名 | 仅展示文件名一行 |

> UI 应同时让用户看到 **纳入方式（原因）** 与 **显示状态（结果）**，避免只有开/关灯而无语义。

### 5.5 目录纳入规则（配置）

对某一文件夹配置（通常通过「应用策略」对话框）：

| 字段 | 说明 |
|------|------|
| 排序字段 | 按名称 / 创建时间 / 更新时间 |
| 排序方向 | 升序 / 降序 |
| 头部数量 | 排序后前 N 个优先 |
| 尾部数量 | 排序后后 M 个优先 |
| 填充策略 | 对其余「自动」文件：不展示 / 仅文件名 / 文件头（非 Markdown 时文件头视为不展示） |

| 操作 | 说明 |
|------|------|
| 应用策略 | 保存到该目录并影响其子项中「自动」文件 |

### 5.6 编辑入口

| 方式 | 说明 |
|------|------|
| 在文件列表行上操作 | 切换纳入、规则开关（主路径） |
| 独立工作树 Tab | 仅预览 + 引导去文件列表改规则（可选） |

## 6. 会话日志（统一时间线）

入口：会话内顶栏 ☰ → **会话操作** → **会话日志**。全屏顶栏标题固定为 **会话日志**（不再单独提供「工具日志」「检查点」页面标题）。

### 6.1 时间线列表

| 功能 | 说明 |
|------|------|
| 统一列表 | 工具执行记录与检查点创建 **同列**、按时间 **倒序**；条目带类型标签（如「工具」「检查点」） |
| 工具条目 | 工具名、成功/失败、简短摘要；左侧色条区分状态 |
| 检查点条目 | 检查点 ID、时间、来源；**回滚到此** 按钮 |
| 关联检查点 | 工具条目上可显示检查点 ID 链接 |
| 数量上限提示 | 页顶说明最多保留 N 个检查点，超出 FIFO 淘汰 |
| 空状态 | 尚无记录时的引导 |

### 6.2 回滚

| 功能 | 说明 |
|------|------|
| 从检查点条目回滚 | 确认后将工作区恢复到此检查点；成功/失败 Toast |
| 从工具条目检查点链接回滚 | 确认后直接回滚对应检查点（不跳转独立检查点页） |
| 回滚中 | 重复操作提示「回滚进行中」 |
| 已淘汰检查点 | 回滚入口禁用或 Toast「检查点已移除」 |

### 6.3 页状态

| 状态 | 说明 |
|------|------|
| 空闲 | 可操作 |
| 回滚中 | 禁用重复回滚 |

## 11. 跨场景用户流程（供流程图参考）

### 11.1 开始写作

1. 选项目 → 选或新建会话 → 进入对话  
2. （可选）从项目模板初始化会话文件  
3. 在「文件」中编辑章节 → 保存  
4. 在对话中让 AI 基于工作树上下文续写  

### 11.2 AI 改文件

1. 用户在对话发送指令  
2. 出现工具调用卡片（进行中 → 成功/失败）  
3. 用户可在「文件」中查看变更；在「日志」中查看详情  
4. 若不满意，从检查点回滚  

### 11.3 调整 AI 可见内容（工作树）

1. 打开工作树预览，确认当前纳入范围  
2. 在文件列表对单文件切换「自动/展示/隐藏」  
3. 对文件夹切换规则开/关或配置目录策略  
4. 回到预览确认文本变化  

### 11.4 模板升级

1. 在上级维护 global 模板  
2. 项目详情执行「从上级同步」→ 警告 → 覆盖  
3. 对已有会话执行「从项目同步」（若提供）→ 警告：会话文件与检查点可能被清空

## 12. 状态与边界（设计需覆盖）

| 场景 | 期望体验 |
|------|----------|
| 无网络 / 服务不可用 | 对话发送失败、模型列表拉取失败的可理解文案 |
| 未选模型 | 无法发送；入口引导去服务商或 Agent |
| 未保存离开编辑 | 确认框：放弃 / 保存 / 取消 |
| 并发回滚 | 第二次操作提示「已有回滚进行中」 |
| 检查点已被 FIFO 淘汰 | 日志中的回滚按钮禁用或说明原因 |
| 导入覆盖 | 强警告 + 二次确认 |
| 长路径 / 长文件名 | 列表截断 + 完整 path 用 title/详情展示 |
| 大文件编辑 | 性能提示或只读建议（产品策略） |
